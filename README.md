To run development:

```
npm install -g serve

```



## Зависимости

* [Node.js](https://nodejs.org)
* [Yarn](https://yarnpkg.com)

## Настройка проекта
```
yarn install
```

### Запус для разработки
```
yarn run serve
```

### Сборка проекта для продакшен
```
yarn run build
```
